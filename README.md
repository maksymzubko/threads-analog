## Description

Is an application similar to Thread, Twitter. Written in JS (nextjs, react).

## Installation

Firstly you need start API, configurate .env file, then:

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# build
$ npm run build
```
